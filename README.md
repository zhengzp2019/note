# mydocs

This project uses MkDocs.

## Install

```bash
python3 -m pip install -r requirements.txt
```

## Local development

```bash
mkdocs serve -f mkdocs.yml
```

## Build

```bash
mkdocs build -f mkdocs.yml
```

## Try different themes

```bash
mkdocs serve -f mkdocs.yml -t material
mkdocs serve -f mkdocs.yml -t terminal
mkdocs serve -f mkdocs.yml -t dracula
mkdocs serve -f mkdocs.yml -t windmill
```
