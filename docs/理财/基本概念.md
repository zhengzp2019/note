# 学习理财

## 概念

- 股票：公司发行的持有公司的凭证。
- 债券：类似于欠条，把钱借给国家，国家打的欠条叫国债；借给公司，公司打的欠条叫企业债。
- 指数：（中国的）沪深300：中国最大的300家公司，中证500：中国中型企业500家，创业板指数：偏科技，创业公司。（美国的）标普500：美国最大的500家公司；纳斯达克100：科技股偏多；道琼斯：30家大型公司；
- 基金：可以理解为把个人钱交给基金经理，由基金经理来帮助个人投资（可以理解为代理，不指代具体的投资放向，例如：股票、债券、指数）

- Q：股票、指数、基金的区别？
  A：股票是单个公司所有权凭证；指数是“一篮子股票”的统计结果；基金是按既定策略去买入资产的投资工具。
- Q：国债和企业债的主要差异？
  A：核心差异是发行主体和信用风险，国债通常风险更低，企业债收益和风险通常更高。
