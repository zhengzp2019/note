# 随心记

现在AI模型的能力已经很强了，在很多方面都已经大幅度的超过了人类，那么如何利用上AI的能力帮助人来更高效的完成各种困难的场景？

现在的Agent可以像人类一样使用工具，它具备自己的大脑和调用工具的能力，那么需要赋予它了解人类的问题，已经对于一个问题应该如何处理的能力？
在我们需要创建一个自己的Agent，首先需要告诉他能够做什么、接着需要告诉他怎么做，对于复杂问题应该采用什么样的流程来处理，即流程编排能力，技术上称之为：[工作流](https://autobots.jd.com/team/a0bb80e4014d4ceb99044b6488e1498e/workflow)。

拥有了工作能力之后，那如何提升AI的工作能力呢？需要从日常的使用反馈中不断的优化AI的能力，即反馈。需要让AI从用户的反馈中知道哪种答案是好的，哪种答案是不好的，从而修改自己的回答策略，使回答能够趋向于用户更希望的方向回答。
